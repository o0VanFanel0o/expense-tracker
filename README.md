💸 Expense Tracker

Aplicación web para registrar gastos, administrar un presupuesto y visualizar datos con una gráfica.

📸 Preview

![Demo](demo.gif)

🚀 Funcionalidades

- Agregar gastos
- Eliminar gastos
- Presupuesto mensual
- Cálculo de total y restante
- Gráfica con Chart.js
- Guardado en localStorage

🛠️ Tecnologías

- HTML
- CSS
- JavaScript
- Chart.js


🧠 Aprendizaje

Este proyecto me ayudó a practicar:

- Manipulación del DOM
- Manejo de eventos
- Uso de localStorage
- Integración de librerías externas
